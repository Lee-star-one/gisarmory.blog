# Leaflet 带箭头的线
使用leaflet.polylineDecorator可实现轨迹箭头展示方向、管道流向箭头展示、河流流向箭头展示等，可动态适配箭头数量，增强可视化展示效果。
使用方式是在正常添加线之后，在添加polylineDecorator即可。
